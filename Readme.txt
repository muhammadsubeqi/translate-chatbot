- menggunakan Speech recognition

Pengenalan ucapan melibatkan penerimaan ucapan melalui mikrofon perangkat, yang kemudian diperiksa oleh Web Speech API terhadap daftar tata bahasa 
(pada dasarnya, kosakata yang ingin Anda kenali di aplikasi tertentu.)
Ketika sebuah kata atau frasa berhasil dikenali, data dikembalikan 
sebagai hasil (atau daftar hasil) sebagai string teks.
